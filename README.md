# 🕒 Gaming Timer (ゲーミングタイマー)

## 1. これは何？（できること）
Gaming Timerは、**ゲーム内の時間（朝・昼・夕・夜など）と現実の時間を連動させて表示するWEBツール**です。
「このゲームは現実の15秒でゲーム内の1分が進む」といった特殊な時間の流れを自動で計算し、現在のゲーム内時間をリアルタイムで表示します。

OBSなどの配信ソフトの「ブラウザソース」として簡単に取り込めるため、ゲーム実況配信などで視聴者に「今のゲーム内時間は〇〇だよ」とおしゃれに表示するのに最適です！

> ⚠️ **ご注意**: このツールを動かすには **WEBサーバー（インターネット上のファイルの置き場所）** が必要です。お手元のパソコン上でダブルクリックしただけではうまく動きません。

---

## 2. 全体的な作業の流れ
このツールをあなたの配信で使うためのステップは以下の通りです。とてもシンプルです！

1. **WEBサーバーを用意する**（無料で使える場所もあります）
2. **ファイルをサーバーにアップロードする**
3. **ブラウザでアクセスして設定・デザインを調整する**
4. **「URLをコピー」して、OBSに貼り付ける**
5. 配信スタート！

---

## 3. フォルダ（ディレクトリ）構成
ダウンロードしたフォルダの中身は以下のようになっています。

```text
GamingTimer/
 ├── index.php         ... メインのプログラムファイル（これをブラウザで開きます）
 ├── README.md         ... この説明書です
 ├── Config/
 │    └── games.csv    ... ゲームの時間の進み方を設定するファイル（メモ帳で編集可能！）
 └── Fonts/
      └── (ここにお好みのフォントファイル[.ttf / .otf]を入れるとWEB上で選べるようになります)
```

## 4. おすすめのレンタルサーバー
ファイルをアップロードして動かすための「WEBサーバー」が必要です。「PHP」というプログラムが動くサーバーを選んでください。

### 🆓 無料で使えるサーバー（まずはお試ししたい方向け）
- **XREA (エクスリア) Free**
  - メリット: 無料で高機能。PHPも動きます。
  - デメリット: パソコン以外（スマホなど）から見た場合、画面に**自動で広告が表示される**ことがあります。
- **スターサーバーフリー (PHP+MySQLプラン)**
  - メリット: 無料でPHPが動きます。
  - デメリット: スマートフォンからのアクセス時に**広告が表示されます**。

### 💰 有料だけど安くておすすめのサーバー（本格的な配信向け）
広告が出ないため、配信画面に余計な広告バナーが映り込む心配がありません。
- **さくらのレンタルサーバ（ライトプラン）**
  - 月額: 約100円〜
  - 特徴: 缶コーヒー1本分の値段で使えて非常に安定しています。広告も一切出ません。迷ったらここが一番確実でおすすめです！
- **ロリポップ！（ライトプラン 以上）**
  - 月額: 約200円〜
  - 特徴: 初心者向けの解説が豊富で画面が使いやすいです。（※一番安いエコノミープランはPHPがうまく使えないことがあるため、必ず「ライトプラン」以上を選んでください）

## 5. 設置（インストール）手順

1. 上記のサーバーを契約し、ファイルをアップロードするための準備（FTPソフトの設定など）をします。（サーバー会社の「ファイルのアップロード方法」というヘルプを参照してください）
2. お手元の `GamingTimer` フォルダの中身（`index.php` や `Config` フォルダなど）を、**まるごとすべて** サーバーにアップロードします。
3. アップロードが終わったら、インターネットブラウザ（ChromeやEdgeなど）で、アップロードした場所のURLを開きます。
   - 例: `https://あなたのドメイン/GamingTimer/index.php`
4. 黒い背景に「GAMING TIMER」という画面が表示されれば設置成功です！

---

## 6. 基本的な使い方 と OBSへの映し方

### STEP 1: ブラウザで設定する
1. ブラウザで設置したURLにアクセスします。
2. **「Game Template」** から、遊びたいゲームを選びます。
3. （手動設定のゲームの場合）**「Current In-Game Time」** に、今のゲーム内の時間を入力します。（例: 朝の8時なら `08:00`）
4. **「Start Clock」** ボタンを押すと、時計が動き始めます！

### STEP 2: デザインを変える（お好みで）
1. 時計が動いている画面の下にある **「⚙️ (歯車マーク)」** ボタンを押します。
2. 文字のフォント、大きさ、文字色、背景色を好きなように変更できます。変更内容は自動で保存されます。

### STEP 3: OBSに表示する（超簡単！）
1. デザインや時間が決まったら、**「🔗 Copy URL」** ボタンを押します。（「✅ Copied!」と出ればコピー完了です）
2. 配信ソフト（OBS Studioなど）を開きます。
3. ソースの `+` ボタンを押し、**「ブラウザ」** を選びます。
4. 設定画面の「URL」の欄に、さきほどコピーしたものを**「貼り付け（ペースト）」** します。
5. OBSの画面に時計が表示されます！ゲームや時間の設定、色などのデザインもすべて完璧に復元された状態で表示されます。

## 7. 新しいゲームを追加したい！（games.csvの修正方法）

リストにないゲームをタイマーに追加したい場合は、`Config` フォルダの中にある `games.csv` というファイルを編集するだけです。プログラミングの知識は全く必要ありません！

### 編集のしかた
1. パソコンの「メモ帳」などのテキストエディタで `Config/games.csv` を開きます。
   - ⚠️ 注意: Excelで開いて保存すると、設定が壊れたり文字化けすることがあるため、**必ず「メモ帳」などのシンプルなテキストエディタ** を使ってください。
2. 以下の書き方のルールに従って、新しい行を一番下に追加して上書き保存します。
3. 保存した `games.csv` を、再びサーバーの `Config/` フォルダへアップロード（上書き）します。

### 書き方のルール
カンマ `,` で区切って、以下の3つの情報を左から順に半角カンマでつないで書きます。

`ゲームの名前, 設定方法などの補足, ゲーム内の「1分」は現実の「何秒」か？`

**【追加の記入例】**
たとえば、「ゲーム内1日が現実の20分」で進むゲームを追加したいとします。
計算すると: 現実の1200秒 ÷ ゲーム内の1日1440分 ＝ **0.8333秒** （ゲーム内の1分＝現実の0.8333秒）となります。

この場合、`games.csv` の一番下の行に以下のように改行して追記します。

```csv
マインクラフト,標準設定,0.8333
農業シミュレーター,倍速モード,1.5
```

- もし、「現実の時間とまったく同じ速度で進むゲーム（どうぶつの森など）」なら、最後の数字を `60` にします。
- これだけで、ブラウザを更新（リロード）すると、プルダウンメニューの一覧に追加したゲームが表示されるようになります！
